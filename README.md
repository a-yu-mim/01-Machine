# 🧮 SPTecher's Machine

A simple numeral base converter that allows conversion between Decimal, Binary, Octal, and Hexadecimal systems.

---

## Features

- Decimal → Binary, Octal, Hexadecimal  
- Binary → Decimal, Octal, Hexadecimal  
- Octal → Decimal, Binary, Hexadecimal  
- Hexadecimal → Decimal, Binary, Octal  

---

## How to use

- Enter a numeric value  
- Click on **Convert**  
- View the result
